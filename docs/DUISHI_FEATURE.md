# 对食功能说明文档

## 功能概述

"对食"是《红楼回忆志》中丫鬟/小厮阶层特有的社交系统。玩家可以与其他丫鬟或小厮建立对食或私约关系，共同收集情报，享受特殊加成。

## 关系类型

### 对食 (dui_shi)
- **正式的关系**：较为正式的伴侣关系
- **情报共享**：双方收集的情报会自动分享给对方
- **特殊事件**：每日可触发特殊的对食事件（交换信物、私下幽会等）
- **双人挂机**：可与搭档一起进行双人挂机，获得额外情报奖励

### 私约 (si_yue)
- **随意的关系**：较为随意的私下约定
- **无情报共享**：不会自动共享情报
- **双人挂机**：仍可享受双人挂机加成

## 如何建立关系

### 1. 发起申请
1. 打开角色信息页面
2. 找到"我的关系"面板
3. 点击"发起对食/私约申请"按钮
4. 选择关系类型（对食或私约）
5. 选择目标角色（仅限丫鬟/小厮阶层）
6. 确认发送申请

### 2. 接受申请
- 当其他玩家向你发送申请后，你会在"待确认申请"列表中看到
- 点击"接受"按钮即可建立关系

### 3. 限制条件
- 只有丫鬟/小厮阶层可以建立对食/私约关系
- 一次只能有一段活跃的关系
- 双方都必须没有现有的关系

## 对食关系功能

### 情报共享
- 当你获得新情报时，会自动分享给对食搭档
- 在关系面板中可以看到共享情报的数量
- 共享的情报会记录在 `shared_intel_ids` 列表中

### 双人挂机
- 在"听壁脚"场景中，如果有对食关系，会显示双人挂机选项
- **加成效果**：
  - 触发率 +20%
  - 情报碎片数量 +1
- 可以选择搭档一起进行挂机

### 特殊事件
每日可触发一次对食特殊事件，事件类型包括：

| 事件名称 | 触发几率 | 效果 |
|---------|---------|------|
| 交换信物 | 15% | 关系加成 +5，情报获取 +10% |
| 私下幽会 | 20% | 关系加成 +3，心情 +10 |
| 分享秘密 | 25% | 关系加成 +8，压力缓解 +5 |
| 维护搭档 | 10% | 体面值 +5，关系加成 +10 |

### 关系对话
系统提供多种对食对话模板，包括：
- 问候对话
- 关怀对话
- 鼓励对话
- 警告对话
- 告别对话

## 背叛机制

### 背叛后果
如果选择背叛对食搭档：
1. **情报泄露**：搭档将获得你们共同收集的所有情报副本
2. **体面下降**：体面值 -10
3. **背叛记录**：忠诚记录增加一次背叛记录
4. **声名狼藉**：累计 3 次背叛将进入"声名狼藉"状态，无法被新主子收留

### 背叛流程
1. 在关系面板点击"背叛搭档"按钮
2. 查看背叛后果说明
3. 输入确认文字"我已知晓后果"
4. 确认背叛

## 数据库结构

### maid_relationships 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| game_id | uuid | 游戏局 ID |
| player_a_uid | uuid | 玩家 A ID |
| player_b_uid | uuid | 玩家 B ID |
| relation_type | maid_relation_type | 关系类型：dui_shi, si_yue |
| status | maid_relation_status | 状态：pending, active, betrayed |
| formed_at | timestamp | 关系建立时间 |
| shared_intel_ids | uuid[] | 共享情报 ID 列表 |
| betrayer_uid | uuid | 背叛者 ID（背叛时记录） |

## 相关脚本

## 开发说明

### 添加新的对食事件
在 `DuiShiEvents.gd` 的 `DUISHI_EVENTS` 字典中添加新事件配置：

```gdscript
"new_event": {
    "name": "事件名称",
    "description": "事件描述",
    "trigger_chance": 0.2,  # 触发几率
    "effects": {
        "effect_name": value
    }
}
```

### 扩展情报共享逻辑
在 `RelationshipManager.gd` 的 `share_intel_with_partner` 函数中扩展共享逻辑。

### 自定义双人挂机加成
在 `EavesdropManager.gd` 中修改以下常量：
- `DUO_BONUS_RATE`：触发率加成
- `DUO_EXTRA_FRAGMENT`：额外情报碎片数量
