### 3. UI 改进

**新增差事分派面板 (`AssignTaskPanel`)**
- 差事类型选择器（OptionButton）
- 目标玩家选择器（OptionButton）
- 赏银输入框（SpinBox）
- 确认/取消按钮

** TreasuryUI.gd 新增函数**
- `_connect_task_panel()` - 连接差事面板信号
- `_update_task_panel_ui()` - 更新差事面板 UI
- `_get_role_display()` - 获取角色类型显示文本
- `_update_reward_input()` - 根据差事类型更新赏银输入范围
- `_on_task_type_selected()` - 差事类型选择回调
- `_on_task_target_selected()` - 目标玩家选择回调
- `_on_task_confirm_pressed()` - 确认分派回调
- `_on_task_cancel_pressed()` - 取消分派回调
- `_execute_assign_task()` - 执行差事分派（带类型参数）

### 4. 数据库更新

**更新 `steward_assign_task` RPC 函数**
- 新增 `p_task_type` 参数（默认 'errand'）
- 根据差事类型动态设置精力消耗和消息内容
- 返回差事类型和实际精力扣除值

**新增 `bulk_assign_tasks` RPC 函数**
- 支持批量分派差事
- 参数：管家 UID、游戏 ID、分派任务数组
- 返回成功/失败统计和详细结果

**迁移文件**: `supabase/migrations/20260319_add_task_types.sql`

---

### 5. 使用流程

1. 打开银库界面（Treasury）
2. 点击"差使分派"按钮
3. 在弹出的差事分派面板中：
   - 选择差事类型（查看精力消耗和基础赏银）
   - 选择目标玩家（显示角色类型）
   - 调整赏银数额（在合理范围内）
4. 点击"确认分派"执行
5. 系统自动扣除管家精力，发放赏银给目标玩家

---

### 6. 注意事项

- 执行差事分派需要管家有足够的精力
- 目标玩家的精力会因差事类型而扣除不同数值
- 赏银有最小值和最大值限制（基础值的 1-3 倍）
- 分派成功后会发送一条消息给目标玩家
